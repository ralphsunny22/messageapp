## About This Project

Simply User Admin Messaging Application, where users can create and view messages sent to a particular admin. Users have limited roles, to certain function. Admin can view messages sent to him specifically. Admin can activate and deactivate a user. Follow basic steps below to install

- npm install
- npm run dev
- setup .env
- php artisan migrate
- php artisan migrate:fresh --seed
- php artisan serve
- Good luck...

Laravel is accessible, powerful, and provides tools required for large, robust applications.
